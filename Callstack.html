Example of Callstack and recursive

It keeps track of functions that are currently being executed and the order in which they were called. When a function is called, it is added to the top of the call stack, and when a function is completed, it is removed from the stack.

A recursive function is a function that calls itself during its execution.Each recursive call processes a smaller instance of the problem until a base case is reached, at which point the function stops calling itself and returns a result.
