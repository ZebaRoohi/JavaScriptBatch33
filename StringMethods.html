String Methods

Concat:You can concatenate strings using the + operator or the concat method.

Length:The length property gives the length of the string.

Accessing Characters:Individual characters can be accessed using square brackets and zero-based indexing.

The substring method extracts a portion of a string.

The indexOf method searches for a substring and returns the index of the first occurrence

The replace method replaces a specified substring with another.

The toUpperCase and toLowerCase methods change the case of the string.

The trim method removes leading and trailing whitespaces from a string.

String Conversion: You can convert values to strings using the String constructor or the toString method.

Negative indices can be used to count from the end of the string
