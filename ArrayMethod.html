Array Methods

Array Literal or array constructor method

Adding Elements:push: Adds one or more elements to the end of an array. unshift: Adds one or more elements to the beginning of an array.

Removing Elements: pop: Removes the last element from an array. shift: Removes the first element from an array.

The concat method creates a new array by combining multiple arrays or values.

The sort() method sorts an array alphabetically.

Returns a shallow copy of a portion of an array.
