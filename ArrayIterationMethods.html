Array Iteration Methods

Map:It is used to iterate over all the elements in an array, which results in a new array

The forEach method in JavaScript is to iterate over elements in an array one by one, without creating a new array.

The filter method in JavaScript is used to create a new array containing elements that pass a certain condition.

find:The find method in JavaScript is used to return the first element in an array that satisfies a given testing function. It stops execution once such an element is found, and it does not iterate over the entire array

The some method is used to check if at least one element in an array satisfies a given condition. It returns true if any element passes the test; otherwise, it returns false

The reduce method in JavaScript is used to reduce the elements of an array to a single value. It takes a callback function and an initial value.
