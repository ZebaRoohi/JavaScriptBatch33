DOM methods

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, and text content. JavaScript can manipulate the DOM dynamically, allowing you to update and interact with the content of a webpage.

1.getElementById method:getElementById is a JavaScript method that is used to select and retrieve an HTML element from the DOM (Document Object Model) based on its unique id attribute


getElementByClassName:The getElementsByClassName method in JavaScript allows you to select and manipulate HTML elements based on their class name.

The getElementsByTagName method in JavaScript allows you to select and manipulate HTML elements based on their tag name.

In JavaScript, query selectors are methods that allow you to select HTML elements from the DOM (Document Object Model) based on various criteria(id,class,tagnames)
