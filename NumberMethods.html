Example of Number methods

ToFixed: rounding the number to a specified number of decimals.

toPrecision(): Formats a number to a specified length.

toString(): Converts a number to a string.

parseInt(): Parses a string and returns an integer.

parseFloat(): Parses a string and returns a floating point number.

isNaN(): Checks whether a value is NaN (Not a Number).

isFinite(): Checks whether a value is finite.

Number.isInteger(): Checks whether a value is an integer.
